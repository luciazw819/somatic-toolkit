# Somatic Toolkit

A personal regulation companion app — somatic exercises, breathing techniques, and grounding flows recommended by my therapist.

## Features

- **Learn** — 14 somatic exercises with step-by-step guidance and practice timers
- **Flows** — Pre-built and custom sequences for specific purposes (morning calm, stress release, wind down)
- **SOS** — Activation-level check with tailored grounding sequences

## Usage

Open [the app](https://yourusername.github.io/somatic-toolkit/) on your phone and add to home screen for an app-like experience.

## Tech

Single-page app built with React, pre-compiled with esbuild. No server needed.
